##  Architecture

- Eureka Service Discovery
- API Gateway
- Microservice Commandes v2 + Produit
- Resilience4j Circuit Breaker
- Load Balancing
- Swagger/OpenAPI

- Eureka, Gateway, Config Server, Resilience4j
- H2 Database, JPA/Hibernate
- Maven, Git/GitHub
